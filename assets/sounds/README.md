# Sound Files

Place the following MP3 files in this folder:

| File | Usage | Recommended style |
|------|-------|-------------------|
| `correct.mp3` | Correct answer in lesson | Short upbeat "ding" (0.5-1s) |
| `incorrect.mp3` | Wrong answer in lesson | Short low "buzz" (0.5-1s) |
| `tap.mp3` | Button/option tap | Very short click (0.1-0.3s) |
| `celebrate.mp3` | Lesson completed | Cheerful fanfare (1-2s) |

Free sources:
- https://mixkit.co/free-sound-effects/game/
- https://freesound.org
