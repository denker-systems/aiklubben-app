# App Icons

Place PNG icon files here. File names must match the icon names in `src/components/ui/AppIcon.tsx`.

## Required icons

### Navigation / Tabs
- `home.png` - Home tab icon
- `news.png` - News tab icon
- `courses.png` - Courses tab icon
- `content.png` - Content/Resources tab icon
- `profile.png` - Profile tab icon

### Menu & Actions
- `settings.png` - Settings icon
- `support.png` - Support/contact icon
- `privacy.png` - Privacy/lock icon
- `about.png` - About/info icon
- `logout.png` - Logout icon

### Stats
- `xp.png` - XP/lightning icon
- `streak.png` - Streak/fire icon

### Course Levels
- `beginner.png` - Beginner level icon
- `intermediate.png` - Intermediate level icon
- `advanced.png` - Advanced level icon

### Content Categories
- `resources.png` - Resources icon
- `platforms.png` - Platforms/brain icon
- `events.png` - Events/target icon

## Recommended size
Export at **96x96px** or **128x128px** for crisp rendering at all sizes.
